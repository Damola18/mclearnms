## MCLEARNMS

  
## Technologies used
* ✅ HTML 
* ✅ CSS
* ✅ JS
* ✅ BOOTSTRAP
* ✅ GSAP(for animations)

## Installation
- Clone the repo and cd into the project folder
- <code>npm start</code> will open the localhost port on which the program is running
## Usage

[Click here](https://www.mclearnms.vercel.app) to open the page in your browser
